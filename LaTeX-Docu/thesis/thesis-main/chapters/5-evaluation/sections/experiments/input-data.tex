\subsection{Experimentübergreifende Eingabedaten}\label{subsec:input-data}

Bei allen durchgeführten Experimenten gibt es einige, gleiche Eingabedaten.
Damit sie nicht redundant wiederholt werden, sind sie hier einmal zusammengefasst.

Einige der überflüssigen Daten aus den Konfigurationen werden hier nicht ausgelistet, damit sie die Lesbarkeit und die Übersichtlichkeit nicht stören.
Beispielsweise wird die Nennung aller hinzugefügten \code{Layer} oder \code{Agenten} aus der Simulationskonfiguration ausgelassen.

\textbf{Simulationskonfiguration:}
\begin{itemize}
    \item \code{startPoint: 2023-06-28T23:59:50}
    \item \code{endPoint: 2023-06-29T23:59:59}
    \item \code{deltaT: 1}
    \item \code{deltaTUnit: seconds}
\end{itemize}

\textbf{\code{HumanTraveler}-Konfiguration:}
\begin{itemize}
    \item \code{hasBike: 0}
    \item \code{hasCar: 0.85}
    \item \code{prefersBike: 0.1}
    \item \code{prefersCar: 0.9}
    \item \code{usesBikeAndRide: 0.2}
    \item \code{usesOwnBike: 0.85}
\end{itemize}

\textbf{\code{BicycleLeader}-Konfiguration:}
\begin{itemize}
    \item \code{hasBike: 1}
    \item \code{hasCar: 0}
    \item \code{prefersBike: 1}
    \item \code{prefersCar: 0}
    \item \code{usesBikeAndRide: 1}
    \item \code{usesOwnBike: 1}
\end{itemize}
